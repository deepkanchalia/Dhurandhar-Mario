# Dhurandhar: The Game

A Super Mario-style platformer featuring Dhurandhar as the hero fighting his way through Level 1.

## How to Play

1. Open `index.html` in your browser (Chrome, Safari, or Firefox)
2. Press SPACE to start
3. Arrow keys to move, SPACE to jump
4. Stomp enemies by jumping on their heads
5. Collect coins for points
6. Reach the flag pole to complete the level

## Controls

| Key | Action |
|---|---|
| Left Arrow | Move left |
| Right Arrow | Move right |
| Space / Up Arrow | Jump |

## Characters

- **Hero**: Dhurandhar (you)
- **Goombas**: Walking enemies (stomp to kill)
- **Koopas**: Turtle enemies (stomp to shell, kick shell)
- **Queen**: Waiting for you at the end

## Troubleshooting

- **Game won't start**: Make sure all images in `assets/` folder are present
- **Blank screen**: Open browser console (Cmd+Option+J on Mac) and check for errors
- **Images not showing**: Don't move index.html away from the assets folder
